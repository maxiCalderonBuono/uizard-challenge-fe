## :muscle: Development

### This project was developed using:

- -> Build tool: Gatsby
- -> Language: Javascript
- -> Framework: React.js
- -> UI: Styled-components
- -> Api management: GraphQl
- -> Deploy: Netlify

## Install

    $ git clone https://github.com/maxiCalderonBuono/uizard-challenge-fe
    $ cd PROJECT
    $ npm install

## Start & watch

    $ gatsby develop

## Simple build for production

    $ gatsby build
